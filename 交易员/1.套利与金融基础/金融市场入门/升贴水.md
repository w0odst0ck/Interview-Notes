# 升贴水 —— 专门用来描述**价差方向**的术语

在套利、期货、ETF、外汇里全都会用到，我用**最清晰、不绕弯**的方式给你讲透。

---

# 1. 一句话定义

**升贴水 = 两个价格之间谁高谁低的状态描述**

最常用场景：

- 现货 vs 期货
- 近月合约 vs 远月合约
- ETF 价格 vs 净值（IOPV/NAV）

---

# 2. 标准公式（记住这个就不会乱）

以 **A 对 B** 来说：

- **A 升水** = A 价格 ＞ B 价格
- **A 贴水** = A 价格 ＜ B 价格

也就是：

**升 = 贵

**贴 = 便宜**

---

# 3. 最常见场景：期货的升贴水

## 期货对现货升贴水

- **期货升水（Contango）**
    
    期货价格 ＞ 现货价格
    
    市场预期未来涨价 / 有仓储资金成本
    
- **期货贴水（Backwardation）**
    
    期货价格 ＜ 现货价格
    
    现货紧张、供不应求
    

## 远月对近月升贴水

- **远月升水**：远月 ＞ 近月（正常结构）
- **远月贴水**：远月 ＜ 近月（现货紧张结构）

---

# 4. 你做套利最常用的：ETF 升贴水

ETF 市价 vs 净值（IOPV/NAV）

- **ETF 溢价（升水）**
    
    市价 ＞ 净值
    
    → 可以申购 ETF → 二级市场卖出套利
    
- **ETF 折价（贴水）**
    
    市价 ＜ 净值
    
    → 二级市场买入 → 赎回套利
    

国内口语里：

**溢价 = 升水

**折价 = 贴水**

---

# 5. 外汇里的升贴水（简单了解）

远期汇率 - 即期汇率

- 远期升水：远期更贵
- 远期贴水：远期更便宜

---

# 6. 升贴水 + 基差 + 价差 的关系（终极总结）

- **价差**：两个价格的差（统称）
- **基差**：现货 − 期货（特殊价差）
- **升贴水**：描述价差是正还是负

用公式串起来：

**基差 = 现货 − 期货**

- 基差 ＞ 0 → 现货升水 / 期货贴水
- 基差 ＜ 0 → 现货贴水 / 期货升水

---

# 7. 套利里怎么用？

看到**大幅升水 / 大幅贴水**

→ 出现套利机会

→ 做收敛交易

→ 回归正常就获利

升贴水一旦回归 0，套利结束。

---

# 8. 超简记忆口诀

**贵 = 升水

**便宜 = 贴水**

**回归 0 = 套利赚钱**
