# 手续费（Commission / Fee）

——**直接决定套利策略能不能赚钱的生死线**

我用**套利 + 量化实战**的方式讲，不废话、全是关键点。

---

# 1. 手续费是什么

你每次买卖资产时，**交易所、券商、监管机构**收的交易成本。

对套利来说：

**手续费 = 固定刚性成本**

赚的价差必须先覆盖它，剩下才是利润。

---

# 2. 手续费的通用结构（所有市场都差不多）

一般由三部分组成：

1. **券商佣金**：券商赚的
2. **交易所规费**：交易所收的
3. **税费**：国家收的（印花税、交易税等）

---

# 3. 各市场手续费水平（套利实战参考）

## A 股股票

- 佣金：万 0.8 ~ 万 2.5，**最低 5 元**
- **印花税：卖出单边 0.1%**（大头）
- 过户费：万 0.01
    
    → **双边成本 ≈ 0.1%+**
    
    → 小额套利基本被 “最低 5 元” 干死

## A 股 ETF

- 佣金：万 0.5 ~ 万 1，很多券商**免 5**
- **无印花税**
    
    → **双边成本 ≈ 0.01%~0.02%**
    
    → 非常适合套利

## 国内期货

- 手续费：**每手几元～十几元**
- 无印花税
- 套利 / 套保单子有**手续费优惠**
    
    → 成本极低，适合高频 / 套利

## 美股 / 美股 ETF / 美期货

- 多数券商：**0 佣金**
- 仅收极小额交易所清算费
    
    → **成本几乎可以忽略**
    
    → 最适合高频、统计套利

## 日股 / 日股 ETF

- 佣金：0.1%~0.3%
- **印花税：卖出单边 0.1%**
    
    → 成本偏高，不适合高频

---

# 4. 关键常识：手续费是双边收取

套利必须**买卖各一次**，所以成本要 ×2：

**总成本 = 买入手续费 + 卖出手续费**

比如 ETF 双边成本 0.02%：

- 价差必须 **>0.02%** 才有可能赚钱
- 再加上滑点，要求更高

---

# 5. 手续费对套利的致命影响

套利赚的就是**几厘、零点几个点**，

手续费稍微高一点，策略直接从赚钱变亏钱。

例子：

- 预期价差收益：0.08%
- 双边手续费 + 滑点：0.1%
    
    → **策略亏损**

所以：

**低手续费市场 = 套利天堂

高手续费市场 = 套利地狱**

# 6. Python 回测怎么写手续费（直接复制）

```python
# 以ETF为例，双边手续费 0.02%
COMMISSION_RATE = 0.0001

# 买入成本
buy_fee = price * amount * COMMISSION_RATE

# 卖出成本
sell_fee = price * amount * COMMISSION_RATE

# 总手续费
total_fee = buy_fee + sell_fee
```

期货按手算：

```python
FEE_PER_LOT = 5     # 每手5元
total_fee = lots * 2 * FEE_PER_LOT
```

# 7. 套利省钱核心技巧

1. **只做低成本品种：ETF、期货、美股**
2. 券商佣金谈到**最低**，ETF 要求**免 5**
3. 避免小额频繁交易（被最低 5 元坑死）
4. 尽量做**持仓少、换手低**的套利策略
