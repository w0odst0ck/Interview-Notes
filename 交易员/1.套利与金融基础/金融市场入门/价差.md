# 1. 价差是什么（一句话）

**两个相关资产价格的差值，就叫价差。**

最常见形式：

Spread=PA​−PB​

套利本质就是：

**价差偏离正常区间 → 入场 → 价差回归 → 获利**

---

# 2. 价差有哪几类（实战套利分类）

## （1）期现价差（现货 — 期货）

也就是你上一个学的 **基差**

- 用于：**期现套利**

## （2）跨期价差（近月合约 — 远月合约）

比如：

- 螺纹钢 2505 合约 − 2510 合约
- ES 标普近月 − 远月

用于：**跨期套利（日历价差）**

## （3）跨品种价差（相关品种之间）

比如：

- 螺纹钢 − 热卷
- 黄金 − 白银
- 恒生科技 ETF − 纳指 ETF

用于：**跨品种套利**

## （4）跨市场价差（同一资产在不同市场）

比如：

- A 股沪深 300 ETF − 新加坡 A50 期货
- 日经 225 期货（日本）− 日经 ETF（美国）

用于：**跨市场套利**

---

# 3. 价差为什么会 “偏离 + 回归”？

- 短期：情绪、资金、流动性、突发事件 → 价差乱走
- 长期：经济逻辑、供需关系、替代关系 → 价差必须回到合理区间

这就是**均值回归**，是套利的数学基础。

---

# 4. 做套利时怎么看待价差？

只看两个状态：

1. **价差过大**
    
    做多便宜的，做空贵的
    
    等价差缩小赚钱
    
2. **价差过小（甚至为负）**
    
    做多贵的，做空便宜的
    
    等价差扩大赚钱
    

一句话：

**套利 = 做价差的均值回归**

---

# 5. 关键概念：价差收敛 vs 发散

- **收敛**：价差回到正常范围 → 套利赚钱
- **发散**：价差越走越偏 → 套利亏钱（风险所在）

所以套利不是无风险，

是**大概率收敛、小概率发散**。

---

# 6. 与基差的区别（非常重要）

- **基差 = 现货 − 期货**（只是一种特殊价差）
- **价差 = 任意两个相关资产价格之差**（广义）

基差 ⊂ 价差

---

# 7. Python 里价差怎么写（直接复制）

```python
# 计算价差
def spread(price_a, price_b):
    return price_a - price_b

# 示例：沪深300ETF vs 股指期货
price_etf = 4.000
price_future = 4020
sp = spread(price_etf * 1000, price_future)
```

做策略常用逻辑：

```python
if sp > upper_threshold:
    # 价差过大，做空A，做多B
    pass
elif sp < lower_threshold:
    # 价差过小，做多A，做空B
    pass
```

---

# 8. 对套利最重要的一句总结

**基差讲的是现货与期货，

价差讲的是一切可比较的两个价格。

套利赚的，全是价差回归的钱。**
