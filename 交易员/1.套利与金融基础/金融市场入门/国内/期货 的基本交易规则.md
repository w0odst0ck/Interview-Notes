期货是**T+0、双向、杠杆、标准化、每日结算**的交易品种，非常适合量化与套利。下面用**AI / 编程 + 套利视角**讲透国内期货核心规则，直接可用于写策略。

---

## 一、交易基础（与股票最大区别）

### 1. 双向交易（可做多 / 做空）

- **做多（Long）**：先买后卖，看涨
- **做空（Short）**：先卖后买，看跌
- 套利意义：**可做价差收敛 / 发散、期现、跨期、跨品种套利**，下跌也能赚钱

### 2. T+0 交易（当日回转）

- 当日开仓 → 当日可平仓，**无限次买卖**
- 套利意义：**日内套利、高频、做市、统计套利**都可行，无隔夜限制

### 3. 标准化合约（交易所统一）

- 每个品种固定：**交易单位、报价单位、最小变动价位、合约月份、最后交易日、交割方式**
- 例：螺纹钢 RB：10 吨 / 手，最小变动 1 元 / 吨，合约月份 1–12 月
- 套利意义：**合约参数固定，可直接写进代码做价差计算**

### 4. 交易时间（含夜盘）

- 日盘：9:00–11:30，13:30–15:00
- 夜盘（多数商品期货）：21:00–次日 02:30（不同品种结束时间不同）
- 金融期货（股指 / 国债）：**无夜盘**
- 套利意义：**夜盘是外盘联动、跨市场套利的关键窗口**

## 二、核心风控制度（套利必懂）

### 1. 保证金制度（杠杆核心）

- 只需缴纳**合约价值 5%–15%** 的保证金即可交易（杠杆 7–20 倍）
- 公式：**交易保证金 = 最新价 × 交易单位 × 手数 × 保证金比例**
- 例：螺纹钢 4000 元 / 吨，10 吨 / 手，保证金 10% → 1 手保证金 = 4000×10×10% = **4000 元**
- 维持保证金：账户可用资金 < 0 → 触发**追加保证金（Margin Call）**，未补足则**强行平仓**中国政府网
- 套利意义：**杠杆放大收益，但必须严格控制仓位；套利策略通常用更低保证金（套保 / 套利优惠）**

### 2. 每日无负债结算（逐日盯市）

- 每日收盘按**结算价**（不是收盘价）清算所有持仓盈亏
- 盈利 → 划入账户；亏损 → 直接扣减保证金
- 账户权益实时更新，**风险不累积**中国政府网
- 套利意义：**每日结算会影响可用资金，套利策略必须考虑结算后的资金占用**

### 3. 涨跌停板制度

- 当日价格波动上限：**± 上一交易日结算价的固定比例**（如螺纹钢 ±7%，原油 ±12%）
- 极端行情（连续涨跌停）：交易所会**扩大涨跌停、提高保证金、强制减仓**
- 套利意义：**涨跌停会导致流动性枯竭，价差可能无法收敛，需设置强平 / 止损**

### 4. 持仓限额与大户报告

- 单个合约**单边持仓上限**（个人 / 机构不同），临近交割月逐步收紧
- 持仓达限额 80% → 必须向交易所**大户报告**
- 套利意义：**大资金套利需拆分账户或申请套保额度，避免超限**

### 5. 强行平仓制度

触发条件：

- 保证金不足且未追加
- 持仓超限额
- 违规交易
- 个人持仓进入交割月未平仓
- 结果：期货公司 / 交易所强制平仓，**亏损由客户承担**中国政府网

### 6. 交割规则（套利收尾）

- 商品期货：**实物交割**，仅**法人可交割**；个人必须在**最后交易日前平仓**
- 金融期货（股指 / 国债）：**现金交割**，按交割结算价轧差清算
- 套利意义：**期现套利最终通过交割收敛；个人套利必须在交割前平仓**

## 三、交易指令与成交规则

### 1. 常用指令

- **限价单（Limit）**：指定价格，仅在更优价成交（套利主力）
- **市价单（Market）**：立即成交，价格不可控（慎用滑点）
- **条件单（Stop）**：触发价到了才下单（适合套利止损 / 止盈）

### 2. 成交原则

- **价格优先 + 时间优先**（同股票）
- **最小变动价位**：必须是其整数倍（如螺纹钢 1 元 / 吨，不能报 4000.5）

---

## 四、交易成本（套利盈亏关键）

- **手续费**：双向收取 = 交易所基础费 + 期货公司加收（可谈低）
- **保证金利息**：占用资金的机会成本
- **交割费用**：实物交割时的仓储、运输、检验费（个人套利无）
- 套利公式：**价差收益 > 双边手续费 + 滑点 + 保证金成本**

## 五、期货 vs A 股 核心对比（套利视角）

|规则|A 股股票|国内期货|对套利影响|
|---|---|---|---|
|交易方向|仅做多|双向（多 / 空）|期货可做全品类套利|
|交易制度|T+1|T+0|期货支持日内套利|
|杠杆|无（融资融券除外）|5%–15% 保证金（7–20 倍）|期货收益 / 风险放大|
|涨跌幅|±10%/20%|±4%–12%（品种不同）|期货涨跌停更严|
|结算|卖出才结算|每日无负债结算|期货每日盯市，风险实时暴露|
|交割|无|实物 / 现金交割|期货期现套利可通过交割收敛|

## 七、适合套利的期货策略（入门）

1. **期现套利**：期货 vs 现货价差回归
2. **跨期套利**：近月 vs 远月合约价差（如买近卖远）
3. **跨品种套利**：相关品种价差（如螺纹 vs 热卷）
4. **统计套利（配对交易）**：两个高度相关品种的价差偏离回归
5. **日内套利 / 做市**：利用盘口价差、挂单吃单
