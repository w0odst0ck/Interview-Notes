# 基差（Basis）

——**期货套利、期现套利的核心指标**

用你能立刻用于套利策略的方式讲清楚。

---

# 1. 基差是什么（一句话定义）

**基差 = 现货价格 − 期货价格**

公式：

Basis=Spot−Futures

它描述的就是：**现货和期货之间的价差**。

---

# 2. 基差的三种状态

1. **基差为正（现货 > 期货）**
    
    叫 **升水（Backwardation）**
    
    → 现货紧俏、供不应求，常见于商品旺季。
    
2. **基差为负（现货 < 期货）**
    
    叫 **贴水（Contango）**
    
    → 远月期货包含仓储、利息、资金成本，最常见。
    
3. **基差接近 0**
    
    → 期现价差合理，无明显套利空间。
    

---

# 3. 基差怎么变？（套利的关键）

基差不是固定的，会随：

- 供需变化
- 资金成本
- 库存
- 季节
- 临近交割

而波动。

**核心规律：越接近交割日，基差越趋近于 0。**

因为到期必须交割，期货必须收敛到现货价格。

这就是**期现套利能赚钱的根本原因**。

---

# 4. 基差与套利的关系（实战逻辑）

## （1）期货价格过高 → 基差偏负

- 现货便宜，期货贵
- 策略：**买现货 + 卖期货**
- 等待基差回归 → 平仓获利

## （2）现货价格过高 → 基差偏正

- 现货贵，期货便宜
- 策略：**卖现货 + 买期货**
- 等待基差回归 → 平仓获利

一句话：

**套利就是赚 “基差从不正常变回正常” 的钱。**

---

# 5. 基差走强 / 走弱

- **基差走强（Stronger Basis）**
    
    变大、变正
    
    对**空头套保、卖期货买现货**有利
    
- **基差走弱（Weaker Basis）**
    
    变小、变负
    
    对**多头套保、买期货卖现货**有利
    

---

# 6. 基差风险（Basis Risk）

你以为会回归，但基差反而扩大 → 亏钱。

这就是**基差风险**，是套利最大的风险之一。

比如：

- 现货涨、期货涨更多 → 基差更负
- 现货跌、期货跌更多 → 基差更正

套利不是无风险，是**低风险 + 基差收敛概率极高**。

---

# 7. Python 里怎么表示基差（直接用）

```python
def calc_basis(spot_price, futures_price):
    return spot_price - futures_price

# 示例
spot = 4000
futures = 4050
basis = calc_basis(spot, futures)  # -50，贴水
```

回测时常用判断：

```python
if basis < -20:   # 期货太贵，开期现套利
    buy_spot()
    sell_futures()
```

---

# 8. 基差 vs 价差（很多人搞混）

- **基差 = 现货 − 期货**
- **价差 = 两个期货合约价格之差（如近月−远月）**

基差 → 期现套利

价差 → 跨期套利、跨品种套利
