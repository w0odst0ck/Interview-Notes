# 一、美股（US Stock）核心规则

### 1. 交易时间（分三段，流动性差异极大）

- **常规交易（Regular Trading Hours, RTH）**
    
    - 美东时间：9:30–16:00（无午休）
    - 北京时间（夏令时 3–11 月）：21:30–04:00
    - 北京时间（冬令时 11–3 月）：22:30–05:00
    - 流动性最好，套利主力时段
    
- **盘前（Pre-Market）**：美东 04:00–09:30
- **盘后（After Hours）**：美东 16:00–20:00
    
    - 盘前 / 盘后流动性差、滑点大，**套利慎用**
    

### 2. 交易制度（T+0 交易，T+2 交割）

- **T+0 回转交易**：当天买入 → 当天卖出，无限次
    
    - 现金账户有 “**日内交易限制**”：5 个交易日内≥4 次日内交易 → 被标记为 “**典型日内交易者（PDT）**”，要求账户净资产≥$25,000
    - 融资账户无此限制
    
- **T+2 交割**：交易后第 2 个工作日完成股票 / 资金交收
    
    - 卖出资金 T+2 才能提现，但**可立即用于再买入**
    

### 3. 交易单位与报价

- **最小交易单位：1 股**（无 “手” 限制，可买 1 股）
- **最小报价单位（Tick Size）**
    
    - 股价≥$1：**$0.01**（1 美分）
    - 股价 <$1：**$0.0001**（0.01 美分）
    
- **无涨跌幅限制**，但有**熔断机制**

### 4. 熔断机制（仅针对下跌）

- **指数熔断（标普 500）**
    
    - 一级：跌 7% → 暂停 15 分钟
    - 二级：跌 13% → 再暂停 15 分钟
    - 三级：跌 20% → 当日停市
    
- **个股熔断（LULD）**
    
    - 5 分钟内波动超 ±5%/10%/20% → 暂停 5 分钟
    - 尾盘 15:25 后豁免
    

### 5. 做空机制（成熟，套利核心）

- 允许**裸做空**（先卖后买），但需**融券**
- 融券规则：**uptick rule（卖空价≥前成交价**
- 对套利意义：**可做配对交易、多空对冲、统计套利**

### 6. 交易成本

- 佣金：多数券商 **$0 佣金**（如 Robinhood、富途）
- 无印花税
- 融券利息：做空成本

# 二、日股（Japan Stock, TSE）核心规则

### 1. 交易时间（两段，有午休）

- **东京时间**
    
    - 早盘：09:00–11:30
    - 午间休市：11:30–12:30
    - 午盘：12:30–15:00
    
- **无盘前 / 盘后交易**
- **开盘前竞价**：08:00–09:00（产生开盘价）
- **收盘竞价**：15:25–15:30（产生收盘价）

### 2. 交易制度（T+0 交易，T+2 交割）

- **T+0 回转**：当天买卖
- **T+2 交割**：与美股一致
- **无日内交易限制**（不像美股 PDT）

### 3. 交易单位与报价（最特殊）

- **最小交易单位：100 股 / 手**（绝大多数股票）
    
    - 例：买 1 手 = 100 股，不能买 50 股
    - 不支持**碎股交易**（<100 股）
    
- **最小报价单位（Tick Size）**：按股价阶梯设置
    
    - 股价 <¥100：¥1
    - ¥100–¥1,000：¥1
    - ¥1,000–¥10,000：¥10
    - ¥10,000+：¥100
    - 对套利：**价差必须是 Tick 的整数倍**
    

### 4. 涨跌幅限制（严格）

- 主板：**±10%**
- 创业板（MOTHERS）：**±20%**
- 新股首日：**±30%**
- 触发即停盘，**无熔断，直接封死**

### 5. 做空机制

- 允许做空，但**融券难度大、成本高**
- 对套利：**做空不如美股方便，多以做多为主**

### 6. 交易成本

- 佣金：约**0.1%–0.3%**（比美股高）
- 印花税：卖出时**0.1%**
- 过户费：约**0.01%**

# 三、美股 vs 日股 vs A 股 核心对比（套利视角）

|规则|美股|日股|A 股|对套利影响|
|---|---|---|---|---|
|交易方向|多空均可|多空均可|仅做多|美 / 日可做对冲套利|
|T+0|是（PDT 限制）|是（无限制）|否（T+1）|美 / 日支持日内套利|
|交易单位|1 股|100 股 / 手|100 股 / 手|日股最小头寸大|
|涨跌幅|无（有熔断）|±10%/20%|±10%/20%|美股波动更大，套利空间大|
|做空|易（融券方便）|难（融券贵）|难（融券少）|美股最适合配对交易|
|盘前 / 盘后|有|无|无|美股可做财报套利|
|成本|极低（0 佣金）|中（有印花税）|中（有印花税）|美股套利成本最低|
|交割|T+2|T+2|T+1|美 / 日资金占用更长|

# 四、套利重点提示

### 美股套利优势

- **T+0 无限制（融资账户）**
- **1 股起买，头寸灵活**
- **做空容易，可做全品类套利**
- **0 佣金，成本极低**
- **适合：日内套利、统计套利、配对交易、期现套利、财报套利**

### 日股套利特点

- **100 股起买，最小头寸大**
- **Tick Size 阶梯，价差计算复杂**
- **做空难，以做多套利为主**
- **适合：跨市场套利（日股 + 美股）、事件套利**
