# 一、美国期货（CME/COMEX/NYMEX）核心规则

### 1. 交易时间（近乎 24 小时，分时段）

- **电子盘（Globex）**：周日 18:00（美中）– 周五 16:15（美中）
    
    - 每日 16:15–18:00 休市 1 小时（系统维护）
    - 北京时间（夏令时）：周一 06:00 – 周六 04:15
    
- **公开喊价（场内）**：仅少数品种保留，流动性低，套利基本不用
- **关键**：**几乎全天可交易**，适合跨时区套利、高频

### 2. 交易制度（T+0、双向、杠杆）

- **T+0 回转**：当天开仓、当天平仓，无限次
- **双向交易**：多空均可，直接做配对 / 对冲套利
- **保证金制度（杠杆）**：
    
    - 初始保证金：约**3%–15%**（品种不同）
    - 维持保证金：低于即**追加 / 强平**
    - 杠杆：**7–30 倍**（常见）
    
- **逐日盯市（MTM）**：每日按结算价清算，**无负债**

### 3. 合约规格（标准化，套利核心）

以主流品种为例：

|品种|合约单位|最小变动（Tick）|每 Tick 价值|
|---|---|---|---|
|美黄金（GC）|100 盎司|$0.1 / 盎司|$10|
|美原油（CL）|1000 桶|$0.01 / 桶|$10|
|标普 500（ES）|$50 / 点|0.25 点|$12.5|
|纳指 100（NQ）|$20 / 点|0.25 点|$5|
|美元指数（DX）|$1000 / 点|0.005 点|$5|

### 4. 涨跌幅与熔断

- **无固定涨跌幅限制**，但有**指数熔断（标普 500）**：
    
    - 7% → 暂停 15 分钟
    - 13% → 再暂停 15 分钟
    - 20% → 全天停市
    
- **个股 / 品种 LULD**：5 分钟波动超阈值 → 暂停 5 分钟

### 5. 交割规则

- **商品期货**：实物交割（黄金、原油等）
- **金融期货**：现金交割（股指、利率等）
- **套利要点**：**到期前必须平仓**，否则进入交割流程

### 6. 交易成本

- 佣金：**$0.5–$2 / 手**（电子盘）
- 无印花税
- 手续费极低，适合高频 / 套利

# 二、日本期货（JPX / 大阪交易所）核心规则

### 1. 交易时间（分时段，有午休）

- **东京时间**：
    
    - 早盘：09:00–11:30
    - 午间休市：11:30–12:30
    - 午盘：12:30–15:00
    
- **无盘前 / 盘后**，仅白天交易
- **开盘前竞价**：08:00–09:00
- **收盘竞价**：15:25–15:30

### 2. 交易制度（T+0、双向、杠杆）

- **T+0 回转**：同美股期货
- **双向交易**：多空均可
- **保证金制度**：
    
    - 初始保证金：约**5%–20%**
    - 杠杆：**5–20 倍**
    
- **逐日盯市**：同美国

### 3. 合约规格（以日经 225 为例）

|品种|合约单位|最小变动（Tick）|每 Tick 价值|
|---|---|---|---|
|日经 225（NK）|1000 日元 / 点|10 点|10,000 日元|
|微型日经（MNI）|50 日元 / 点|1 点|50 日元|
|日元期货（JPY）|1250 万日元|0.01 点|1250 日元|

### 4. 涨跌幅限制（严格）

- **主板品种**：**±10%**
- **创业板 / 高波动**：**±20%**
- **触发即停盘**，无熔断，直接封死

### 5. 交割规则

- **日经 225 期货**：现金交割
- **商品期货**：实物交割
- **套利要点**：到期前平仓，避免交割

### 6. 交易成本

- 佣金：约**0.1%–0.3%**（比美国高）
- 有**交易税**（约 0.01%）
- 成本高于美国，高频套利性价比低

# 三、美日期货 vs A 股期货 核心对比（套利视角）

|规则|美国期货|日本期货|A 股期货|对套利影响|
|---|---|---|---|---|
|交易时间|近乎 24 小时|白天 4 小时|白天 4 小时|美期货可跨时区套利|
|T+0|是|是|是|均支持日内套利|
|双向交易|是|是|是|均可做对冲 / 配对|
|杠杆|7–30 倍|5–20 倍|5–10 倍|美期货杠杆更高|
|涨跌幅|无（有熔断）|±10%/20%|±5%/10%|美期货波动更大|
|交易单位|1 手起（标准化）|1 手起（标准化）|1 手起|均标准化，易套利|
|成本|极低|中|中|美期货套利成本最低|
|交割|实物 / 现金|实物 / 现金|实物 / 现金|均需到期前平仓|

# 四、套利重点提示

### 美国期货套利优势

- **24 小时交易**，适合跨市场、跨时区套利
- **杠杆高、成本低**，适合高频、统计套利
- **品种丰富**（股指、商品、外汇、利率），可做全品类套利
- **适合**：期现套利、跨期套利、跨市场套利、高频做市

### 日本期货套利特点

- **白天交易**，与 A 股 / 港股时间重叠，适合跨市场套利
- **涨跌幅严格**，波动相对可控
- **品种集中**（日经 225 为主），适合指数套利
- **适合**：日经 - 美股指数套利、事件套利
