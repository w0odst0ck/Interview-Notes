# 三大基础原则・均值回归

——**所有统计套利、价差套利、期现套利的底层逻辑**

你前面学的基差、价差、对冲、风控，全都围绕它展开。

---

# 1. 一句话定义

**均值回归（Mean Reversion）：

**价格或价差偏离长期平均水平后，有极高概率重新回到平均值附近。**

简单说：

- 涨多了会跌回来
- 跌多了会涨回去
- 价差扯太开，会自动 “收回去”

套利赚的就是：

**从偏离 → 回到均值** 这一段的钱。

---

# 2. 为什么会回归？（金融本质）

两个高度相关的资产，比如：

- 沪深 300 ETF vs 股指期货
- 近月期货 vs 远月期货
- 同行业两只龙头股
- 现货 vs 期货

它们的**经济关系、供需、资金成本、替代关系**是稳定的，

短期情绪、资金、流动性会让它们偏离，

但长期不可能一直离谱，否则就会有大量套利资金冲进来抹平价差。

**套利盘本身，就是推动均值回归的力量。**

---

# 3. 套利里的 “均值” 指什么？

不是股价的均值，而是：

- 基差均值
- 价差均值
- 折溢价率均值
- 相关性资产的比值均值

我们做策略永远看的是：

**spread = P₁ − P₂ 的均值**

而不是单独某一个价格。

---

# 4. 最经典的策略逻辑（直接能写代码）

1. 计算价差的**均值 μ**
2. 计算价差的**标准差 σ**
3. 设定上下轨：
    
    - 上轨：μ + kσ
    - 下轨：μ − kσ
    
4. 触发规则：
    
    - 价差 **> 上轨** → 做空 A、做多 B
    - 价差 **< 下轨** → 做多 A、做空 B
    
5. 回归到均值附近 → 平仓获利

这就是 **Z-score 配对交易**的核心。

---

# 5. 均值回归的三个关键特点

1. **胜率极高**
    
    大部分时间都在回归，所以套利稳定赚钱。
    
2. **尾部风险（黑天鹅）**
    
    少数时候价差不回归、反而发散 → 亏钱
    
    这就是必须**风控 + 止损**的原因。
    
3. **收益有限但稳定**
    
    回到均值就结束，不会像趋势那样一直赚
    
    靠**多次、薄利、复利**赚钱。
    

---

# 6. 和你前面学的知识点串联

- **基差** → 均值回归 → 期现套利
- **价差** → 均值回归 → 跨期 / 跨品种套利
- **升贴水** → 回归到 0 → ETF 套利
- **双向对冲** → 只赚回归，不赌方向
- **单笔风控** → 防止不回归时爆仓

一句话：

**套利 = 寻找偏离 + 双向对冲 + 等待均值回归 + 止盈离场**
