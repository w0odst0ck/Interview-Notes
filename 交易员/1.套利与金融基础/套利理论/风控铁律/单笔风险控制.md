# 风控铁律：单笔风险控制

（套利、量化、交易通用，**保命级规则**，直接给你能写进代码的硬标准）

---

# 1. 什么是 “单笔风险”

单笔风险 = **这一单如果做错、止损被打掉，最多亏多少钱**

核心思想：

**任何一笔交易，都不允许亏到影响你下一次交易。**

套利不是无风险，只是概率高，**黑天鹅一来就清零**。

---

# 2. 行业通用铁律（全球机构统一标准）

## 单笔交易最大风险

**不超过账户总资金的 1%**

进阶保守版（套利推荐）：

**单笔风险 ≤ 0.5% 总资金**

极端保守（高频 / 套利）：

**单笔风险 ≤ 0.1%~0.2%**

> 你有 100 万
> 
> 单笔最多亏：> 1% → 1 万
> 
> 0.5% → 5000

连续亏 20 次才亏 20%，不会爆仓。

---

# 3. 为什么必须卡死这条？

- 套利策略靠**胜率高、薄利多次**赚钱
- 一次大亏就能把几十次利润吃光
- 市场跳空、涨跌停、流动性枯竭时，止损会被击穿
- 不控制单笔风险 = 一次翻车直接退出市场

---

# 4. 怎么计算单笔仓位（实战公式）

公式：

最大仓位=每单位止损金额总资金×单笔风险比例​

举例：

- 总资金：100 万
- 单笔风险：0.5% → 5000 元
- 计划止损：价差 0.2 元
- 每份 / 每手亏损：0.2 元

最大仓位 = 5000 / 0.2 = **25000 份**

超过这个仓位，就是**违规风控**。

---

# 5. 套利里的特殊风控：不止损 = 爆仓

很多人觉得套利是 “无风险”，不止损，结果：

- 基差 / 价差持续发散
- 杠杆放大亏损
- 保证金不足被强平
- 直接亏光

**套利必须设止损，而且必须严格单笔风险。**

---

# 6. 风控铁律总结（背下来）

1. **单笔最大亏损 ≤ 总资金 1%**
2. 套利建议更严：**≤ 0.5%**
3. 先算风险，再算仓位
4. 止损一旦设定，绝不移动
5. 不因为 “这次肯定稳” 就破例

---

# 7. 可直接写进 Python 的风控常量

```python
# 单笔风险控制（核心）
ACCOUNT_BALANCE = 1000000    # 账户总资金
SINGLE_TRADE_RISK = 0.005    # 单笔 0.5%
MAX_SINGLE_LOSS = ACCOUNT_BALANCE * SINGLE_TRADE_RISK

# 计算最大仓位
stop_loss_per_unit = 0.2
max_position = MAX_SINGLE_LOSS / stop_loss_per_unit
```
