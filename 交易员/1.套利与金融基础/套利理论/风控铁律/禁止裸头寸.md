# 风控铁律：禁止裸头寸

这是**套利、对冲交易里最核心、最要命的一条铁律**，比止损还重要。

我用最直白、不绕弯的方式讲清楚。

---

# 1. 什么是裸头寸？

**裸头寸 = 只有单边持仓，没有对应的对冲盘。**

简单说：

- 只买不卖
- 只卖不买
- 只多不空
- 只空不多

这就叫**裸奔**，所以叫**裸头寸**。

---

# 2. 为什么套利绝对禁止裸头寸？

套利的本质是：

**赚价差 / 基差收敛的钱，不赌涨跌。**

一旦变成裸头寸：

- 你就从**套利者**变成了**投机赌徒**
- 市场一反向，直接大亏
- 黑天鹅一来，瞬间爆仓

套利策略的核心是：

**对冲掉涨跌风险，只赚价差的钱。**

裸头寸 = 放弃对冲 = 放弃风控。

---

# 3. 举个最直观的例子

### 正常套利（合规）

- 买 A
- 卖 B
    
    → 涨跌跟你没关系，只赚 A-B 价差收敛

### 裸头寸（违规）

- 只买 A，没卖 B
    
    → 大盘一跌，你大亏
- 只卖 B，没买 A
    
    → 大盘一涨，你大亏

这就是裸头寸的风险。

---

# 4. 再比如期货期现套利

正确：

- 买现货
- 空期货
    
    → 无论涨跌都安全，只赚基差

裸头寸错误：

- 只空期货，不买现货
    
    → 行情暴涨直接穿仓

---

# 5. 裸头寸最危险的地方

很多人一开始是套利，

做着做着觉得 “方向很明显”，

就把对冲的一边平掉，变成**裸头寸**。

这是**90% 套利账户爆仓的真实原因**。

---

# 6. 这条铁律的终极一句话

**套利只做对冲，不做方向。

任何单边敞口，都必须立刻对冲掉。

永远不允许裸头寸存在。**

---

```python
# 风控检查：多空必须成对
long_pos = get_long_position()
short_pos = get_short_position()

# 裸头寸判断
if abs(long_pos - short_pos) > threshold:
    cancel_all_orders()
    close_position()    # 强制平仓
    alert("出现裸头寸，策略停止！")
```
