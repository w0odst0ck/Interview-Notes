# 一、套利的核心定义与市场逻辑

1. 套利是交易利用两个或多个高度相关资产之间因市场摩擦、流动性差异、资金不对等等原因产生的**短期价格偏离**，通过**低买高卖 + 双向对冲**的方式，捕捉价差回归带来的低风险收益。套利不预判方向，不博弈单边行情，核心赚取定价修复与均值回归的收益。

# 二、套利的三大基础原则

1. **相关性原则**：交易标的必须具备高度相关性（历史走势高度同步）。
2. **对冲原则**：同时建立多空双向头寸，对冲系统性风险（大盘波动风险）。
3. **均值回归原则**：偏离合理区间的价差，大概率会在有限时间内回归。

# 三、四类主流专业套利模式

1. **跨品种套利**：利用产业链或替代关系高度相关品种进行套利。例：黄金 - 白银、豆油 - 棕榈油、螺纹钢 - 铁矿石。逻辑：价差长期稳定，短期偏离后回归回归。
2. **跨期套利**：同一标的、不同交割月份合约之间的价差套利。例：近月合约与远月合约的价差偏离。逻辑：价差受持有成本、供需预期驱动，具备稳定区间。
3. **ETF 期现套利（指数套利）**：利用 ETF 价格与 IOPV（实时参考净值）净值、股指期货与现货指数的价差套利。逻辑：定价一定律 —— 同一资产在不同市场应价格一致。
4. **统计套利（配对交易）**：基于历史价格规律，对具备协整关系的资产进行低买高卖。逻辑：价格长期同步，短期偏离后会快速修复。

# 四、套利交易的完整执行流程

1. **价差监控**：判断当前价差是否处于历史极端区间。
2. **信号确认**：价差偏离达到阈值，产生入场信号。
3. **对冲开仓**：同时买入低估、卖出高估资产。
4. **持仓等待**：持有至价差回归合理区间。
5. **同步平仓**：双边同时平仓，锁定套利收益。
6. **风险控制**：价差异常扩大时严格止损。

# 五、专业套利风控铁律

1. 必须双向对冲，禁止裸头寸。
2. 单笔套利风险 ≤ 账户总资金 1%-2%。
3. 不做相关性低、流动性差的品种。
4. 必须计算手续费、冲击成本、滑点。
5. 价差不回归不扛单，严格执行止损。
6. 分散多组套利，避免单一策略集中风险。

# 六、套利收益来源

1. 定价偏差修复收益
2. 市场情绪与资金冲击带来的短期无效性
3. 不同市场、不同合约间的流动性溢价
4. 统计规律下的均值回归收益

# 七、套利与普通趋势交易的核心区别

1. **趋势交易**：赚方向收益，承担高波动风险。
2. **套利交易**：赚价差收益，对冲系统风险，收益更稳定。

# 八、总结

套利是利用市场短期无效性赚取低风险价差回归收益，依靠相关性、对冲、均值回归与严格风控实现长期稳定盈利。
