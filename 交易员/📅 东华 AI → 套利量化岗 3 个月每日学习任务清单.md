## 第一阶段：套利与金融基础（第 1-4 周）

### 第 1 周：吃透套利核心逻辑

- **Day1-2**：精读你手上的套利文档，逐段梳理三大原则（相关性 / 对冲 / 均值回归）+ 四类套利模式，整理成思维导图。
- **Day3-4**：阅读《量化投资策略与技术》套利章节，重点对比 “趋势交易 vs 套利交易” 的区别，做 1 页笔记。
- **Day5**：B 站学习「期货 / ETF 交易规则入门」，了解合约规格、涨跌幅、交割规则，记录 3 个核心概念（基差 / 滑点 / 手续费）。
- **Day6-7**：Python 入门金融数据处理：用 Pandas 加载一段期货 / ETF 历史数据，计算收盘价、价差，用 Matplotlib 画出走势。

### 第 2 周：金融市场认知强化

- **Day1-2**：学习国内期货品种（螺纹钢 / 铁矿石 / 黄金）、ETF（沪深 300 / 中证 500）的交易时间、保证金比例、最小变动价位。
- **Day3-4**：理解 “价差偏离 - 均值回归” 的直观逻辑，找一组历史价差数据（如黄金 - 白银），手动计算均值、标准差，标记极端区间。
- **Day5-6**：用 Python 实现价差可视化函数，输入两个品种代码，自动画出价差曲线 + 均值线 +±2σ 区间。
- **Day7**：复盘本周内容，默写套利三大原则 + 风控铁律，确保能脱稿解释。

### 第 3 周：Python 金融数据实战

- **Day1-2**：学习 AkShare/Tushare 库，编写函数获取期货 / ETF 日频数据，保存为 CSV 格式。
- **Day3-4**：计算两组品种的相关系数（Pandas.corr），筛选出相关性 > 0.8 的配对（如豆油 - 棕榈油）。
- **Day5-6**：用 Python 计算配对品种的价差序列，做 ADF 平稳性检验（statsmodels），判断是否具备均值回归特征。
- **Day7**：整理本周代码，上传到 GitHub，写 README 说明 “数据获取→相关性计算→平稳性检验” 流程。

### 第 4 周：阶段复盘与小测

- **Day1-3**：回顾前 3 周内容，查漏补缺：重点巩固套利逻辑、金融概念、Python 数据处理。
- **Day4-5**：小测：随机选一组品种，完整走一遍 “获取数据→计算相关性→计算价差→可视化→平稳性检验” 流程。
- **Day6-7**：整理第一阶段学习笔记，形成《套利基础手册》，为下一阶段统计建模做准备。

## 第二阶段：统计与量化建模（第 5-10 周）

### 第 5-6 周：统计套利核心工具

- **Day1-3**：学习时间序列平稳性（ADF 检验）、协整检验（Johansen）原理，看 B 站「配对交易数学基础」教程。
- **Day4-6**：用 Python 实现 ADF 检验和 Johansen 协整检验，对筛选出的配对做验证，记录检验结果。
- **Day7-10**：学习误差修正模型（ECM），理解 “价差偏离后如何向均值修复” 的数学表达，编写 ECM 模型代码。

### 第 7-8 周：Python 量化回测框架

- **Day1-3**：学习 Backtrader/VectorBT 回测框架，搭建基础回测环境，跑通官方示例策略。
- **Day4-6**：实现简单配对交易策略：价差上穿 + 2σ 时做空高价品种、做多低价品种；价差回归均值时平仓。
- **Day7-10**：在回测中加入手续费、滑点模拟，计算策略收益、最大回撤、夏普比率，优化参数（如均值窗口、σ 倍数）。

### 第 9-10 周：C++ 量化方向补强

- **Day1-3**：复习 C++ 核心知识点：指针、内存管理、STL 容器、多线程编程，刷 LeetCode C++ 基础题。
- **Day4-6**：阅读《High-Frequency Trading》低延迟章节，理解量化系统中 “低延迟” 的实现逻辑（如内存池、无锁队列）。
- **Day7-10**：用 C++ 编写一个简单的行情数据处理程序，模拟接收 tick 数据、计算价差，为后续低延迟策略做铺垫。

## 💡 每日执行小贴士

1. **固定学习时段**：建议每天晚 8-10 点，专注 2 小时，避免碎片化。
2. **输出驱动学习**：每学完一个知识点，必须写代码 / 做笔记 / 可视化，拒绝 “只看不动手”。
3. **项目优先**：所有学习都要服务于最终的 3 个套利项目，带着项目目标去学效率更高。
4. **利用 AI 优势**：把机器学习知识融入策略（如用 ML 预测价差回归时间），在面试中形成差异化竞争力。
