### Pandas 常用模板与方法

#### 1. 数据读取

```python
import pandas as pd

# 从CSV文件读取数据
data = pd.read_csv('data.csv')
# 从Excel文件读取数据
data = pd.read_excel('data.xlsx')
```

#### 2. 数据查看

```python
# 查看前几行
data.head()
# 查看后几行
data.tail()
# 查看数据信息
data.info()
# 查看数据统计摘要
data.describe()
```

#### 3. 数据选择

```python
# 选择单列
col = data['column_name']
# 选择多列
cols = data[['col1', 'col2']]
# 根据条件选择行
filtered_data = data[data['column'] > 10]
```

#### 4. 数据清洗

```python
# 删除含有缺失值的行
data.dropna(inplace=True)
# 填充缺失值
data.fillna(value=0, inplace=True)
```

#### 5. 数据转换

```python
# 更改数据类型
data['column'] = data['column'].astype('int')
# 应用函数到列
data['new_col'] = data['old_col'].apply(lambda x: x * 2)
```

#### 6. 数据分组与聚合

```python
# 按某列分组并计算均值
grouped = data.groupby('category')['value'].mean()
```

#### 7. 数据合并

```python
# 横向合并两个DataFrame
merged = pd.merge(df1, df2, on='key')
# 纵向合并两个DataFrame
concatenated = pd.concat([df1, df2])
```

#### Pandas 常用方法总结

- **数据输入输出**：`read_csv`、`read_excel`、`to_csv`、`to_excel`
- **数据查看**：`head`、`tail`、`info`、`describe`
- **索引与选择**：`loc`、`iloc`、`ix`（较旧，不推荐）
- **数据清洗**：`dropna`、`fillna`、`replace`
- **数据转换**：`astype`、`apply`、`map`
- **分组与聚合**：`groupby`、`agg`
- **数据合并**：`merge`、`concat`

### Numpy 常用模板与方法

#### 1. 创建数组

```python
import numpy as np

# 创建一维数组
arr1d = np.array([1, 2, 3])
# 创建二维数组
arr2d = np.array([[1, 2], [3, 4]])
# 创建全零数组
zeros_arr = np.zeros((3, 3))
# 创建全一数组
ones_arr = np.ones((2, 2))
# 创建指定范围的数组
range_arr = np.arange(0, 10, 2)
```

#### 2. 数组运算

```python
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])

# 加法
add_result = arr1 + arr2
# 乘法
multiply_result = arr1 * arr2
```

#### 3. 数学运算

```python
arr = np.array([1, 2, 3])
# 计算平方
square_result = np.square(arr)
# 计算平方根
sqrt_result = np.sqrt(arr)
# 计算总和
sum_result = np.sum(arr)
```

#### 4. 数组形状操作

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])
# 重塑数组
reshaped_arr = arr.reshape((3, 2))
# 转置数组
transposed_arr = arr.T
```

#### 5. 条件筛选

```python
arr = np.array([1, 2, 3, 4, 5])
filtered_arr = arr[arr > 3]
```

#### Numpy 常用方法总结

- **数组创建**：`array`、`zeros`、`ones`、`arange`、`linspace`
- **数学运算**：`add`、`subtract`、`multiply`、`divide`、`square`、`sqrt`、`sum`、`mean`、`std`
- **形状操作**：`reshape`、`ravel`、`transpose`（或`T`）
- **索引与筛选**：基本索引、布尔索引
