# XGBoost / LightGBM / CatBoost 核心思想（面试**一句话封神版**）

**核心思想：串行训练、降低残差**

我给你拆成**最直白、能直接口述**的版本：

## 1. 串行训练（Serial）

- **不是一起训练**，是**一棵接一棵**训练
- 第 1 棵树先学 → 预测错了
- 第 2 棵树**专门去学上一棵树的错误**
- 第 3 棵树再学前两棵的错误
- …… 一直叠加，直到误差足够小

## 2. 降低残差（Reduce Residual）

- 每棵树不学原始目标 y，而是学**残差（真实值 - 预测值）**
- 目标：**让整体预测越来越准，残差越来越小**
- 本质就是**梯度下降**在函数空间做优化（GBDT）

# 终极背诵（背这一段就够）

**Boosting 家族（XGB/LGB/Cat）采用串行训练，每棵树专门学习前面模型的残差，不断修正错误、降低整体偏差，最终叠加成高精度强模型。**
