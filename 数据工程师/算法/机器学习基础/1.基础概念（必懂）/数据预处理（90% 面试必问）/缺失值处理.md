# 缺失值处理（数据分析师面试极简版）

## 一、常见处理方法（必背）

1. **删除法**
    
    - 删除含缺失值的**行 / 列**
    - 适用：缺失极少、变量不重要
    
2. **填充法（最常用）**
    
    - 数值型：**均值、中位数、众数**
    - 分类型：**众数、固定值（如 Unknown）**
    - 时序数据：**前向填充 / 后向填充**
    
3. **模型填充**
    
    - KNN、随机森林等预测缺失值
    - 适用：缺失较多、数据质量高
    
4. **标记缺失**
    
    - 新增一列`is_missing=0/1`
    - 告诉模型**缺失本身有信息**
    

## 二、一句话原则

- 缺失**少**：删或简单填充
- 缺失**多**：模型填充或标记
- 时序 / 业务有意义：**优先保留缺失信息**
