# 特征构造（面试 + 实战高频）｜极简背诵版

**一句话：从原始字段里 “造” 更有预测力的新特征，是提分关键。**

## 一、最常用 6 大类构造方法（背这些就够）

### 1. 统计特征

- 求和、均值、中位数、最大 / 最小、标准差、计数
- 例：用户近 7 天**平均消费**、**最大单笔金额**

### 2. 时间特征

- 小时、星期、月份、是否周末、是否节假日
- 时间差：间隔天数 / 小时、距上次行为间隔
- 例：距离上次登录天数、注册时长

### 3. 比率 / 比例特征（**非常强**）

- 占比、完成率、转化率、增长率
- 例：点击 / 曝光、购买 / 加购、逾期 / 总借款

### 4. 交叉组合特征

- 特征 A × 特征 B、A / B、A + B、A - B
- 例：年龄 × 收入、消费 / 收入

### 5. 离散化 / 分桶后特征

- 连续值分桶后变成类别或计数
- 例：低 / 中 / 高消费人群

### 6. 历史行为聚合（业务常用）

- 近 N 天、近 M 次、累计、环比、同比
- 例：近 30 天消费趋势↑↓

---

## 二、一句话背诵

**特征构造就是造特征：统计、时间、比率、交叉、分桶、历史聚合，大幅提升模型效果。**
