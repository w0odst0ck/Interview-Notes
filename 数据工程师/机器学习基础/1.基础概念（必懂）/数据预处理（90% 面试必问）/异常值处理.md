# 异常值处理（数据分析师面试极简版）

## 一、怎么发现

1. **描述统计**：看最大 / 最小值、均值 / 中位数差距
2. **可视化**：箱线图、直方图、散点图
3. **统计方法**：Z 分数、IQR 四分位距

## 二、常用判断方法

1. **Z 分数法**
    
    - 通常 |Z| > 3 视为异常
    
2. **IQR 法（箱线图）**
    
    - 小于 Q1−1.5IQR 或 大于 Q3+1.5IQR 为异常
    

## 三、怎么处理（面试必背）

1. **删除**
    
    - 异常少、明显错误、无业务意义
    
2. **替换 / 修正**
    
    - 用**均值、中位数、众数**替换
    - 或按业务逻辑修正（如年龄 199→19）
    
3. **分箱 / 盖帽**
    
    - 上下限截断（capping）：超过阈值统一设为临界值
    
4. **单独建模**
    
    - 异常本身有价值（如欺诈、流失），单独分析
    

## 一句话总结

**先识别（统计 / 画图），再处理（删 / 替 / 截 / 单独用），优先结合业务判断。**

# Z 分数法 & IQR 法（异常值判断）极简解析

## 1. Z 分数法（Z-score）

### 是什么

把数据**标准化**，看一个点**离均值有多远**（用标准差衡量）。

公式：

$Z=\frac{x−μ​}{\alpha}$

- Z：标准化后的值
- μ：均值
- σ：标准差

### 为什么 |Z| > 3 算异常

正态分布下：

- 约 **99.73%** 数据在 ∣Z∣≤3 内
- 超过的只有 **0.27%**，属于**极端小概率**，视为异常

**一句话**：

**Z 分数看偏离均值多少个标准差，超过 3 个标准差就是异常。**

---

## 2. IQR 法（四分位距法，箱线图原理）

### 先懂三个数

- Q1：第 25 分位数（下四分位数）
- Q2：中位数（第 50 分位数）
- Q3：第 75 分位数（上四分位数）

### IQR 是什么

**IQR = Q3 − Q1**

中间 50% 数据的跨度。

### 异常值判断规则

- 小于 **Q1 − 1.5×IQR** → 下异常
- 大于 **Q3 + 1.5×IQR** → 上异常

这就是箱线图里**画圈的那些点**。

**一句话**：

**IQR 法用四分位距定边界，超出 1.5 倍 IQR 范围就是异常。**

---

# 超简对比（面试必背）

- **Z 分数**：基于**正态分布**，适合近似正态的数据
- **IQR**：基于**分位数**，**不受极端值影响**，更稳健
