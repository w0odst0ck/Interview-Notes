# 数据分析师必备机器学习基础（极简面试版）

## 一、基础概念（必懂）

1. **机器学习定义与分类**
    
    - 监督学习、无监督学习、半监督、强化学习（知道区别即可）
    
2. **核心术语**
    
    - 特征 / 标签 / 样本 / 训练集 / 测试集 / 验证集
    - 过拟合 & 欠拟合（怎么判断、怎么解决）
    - 偏差 vs 方差（Bias-Variance Tradeoff）
    
3. **数据预处理（90% 面试必问）**
    
    - 缺失值处理：删除、均值 / 中位数 / 众数填充、模型填充
    - 异常值处理：3σ、IQR、箱线图
    - 特征编码：One-hot、LabelEncoder、OrdinalEncoder
    - 特征缩放：标准化 (StandardScaler)、归一化 (MinMaxScaler)
    - 数据划分：train_test_split、分层抽样 stratify
    

## 二、监督学习（核心必考）

### 1. 分类算法（分析师最常用）

- **逻辑回归 Logistic Regression**
    
    - 原理（线性回归 + sigmoid）
    - 适用场景、优缺点
    - 输出是概率，可做排序、评分
    
- **决策树 Decision Tree**
    
    - 分裂准则：ID3 (信息增益)、C4.5 (增益比)、CART (Gini)
    - 剪枝（预剪枝、后剪枝）
    
- **随机森林 Random Forest**
    
    - 集成思想：Bagging + 随机特征
    - 为什么效果好、抗过拟合
    - 重要特征输出（feature_importances_）
    
- **XGBoost / LightGBM / CatBoost**（华为 OD 高频）
    
    - 属于 Boosting
    - 核心思想：串行训练、降低残差
    - 优缺点、适用数据规模
    - 常用调参：learning_rate、max_depth、n_estimators
    

### 2. 回归算法

- 线性回归
- 岭回归、Lasso（正则化解决过拟合）
- 回归评价指标：MAE、MSE、RMSE、R²

## 三、无监督学习（必考简单版）

- **K-Means**
    
    - 原理、步骤
    - 优缺点、K 值怎么选（肘部法则、轮廓系数）
    
- **聚类评价**：轮廓系数、CH 指数
- **PCA 降维**
    
    - 作用：去冗余、降维、可视化
    - 不适合解释性强的业务分析
    

## 四、模型评估与验证（必背）

### 分类指标

- 精确率、召回率、F1、AUC、ROC
- 混淆矩阵（TP/TN/FP/FN）
- PR 曲线 vs ROC 曲线（不平衡数据用 PR）

### 验证方法

- 留出法
- K 折交叉验证（KFold/StratifiedKFold）

## 五、特征工程（数据分析师灵魂）

- 特征构造：时间特征、统计特征、交叉特征
- 特征选择：
    
    - 过滤法（相关系数、卡方）
    - 包装法（递归消除）
    - 嵌入法（树模型特征重要性）
    
- 数据 leakage 数据泄露（高频坑题）

## 六、业务落地能力（面试官最爱问）

- 怎么把模型结果落地成报表 / 策略 / 评分卡
- 如何向业务解释模型（不是甩 AUC，而是讲影响）
- AB 实验基础知识（对照组、流量分配、显著性）

## 七、Python 工具链（必须会写代码）

- numpy、pandas（数据处理）
- matplotlib/seaborn（可视化）
- scikit-learn（全套建模）
- xgboost/lightgbm（提升树）
- 能**独立跑完一整套流程**：
    
    读数据 → 清洗 → 特征工程 → 划分 → 训练 → 评估 → 输出结果
