# 回归四大评价指标（MAE / MSE / RMSE / R²）

## 1. MAE（平均绝对误差）

$MAE = \frac{1}{n} \sum |y_i - \hat{y}_i|$

- **含义**：平均**绝对误差**
- **特点**：直观、稳健，**对异常值不那么敏感**

## 2. MSE（均方误差）

$MSE = \frac{1}{n} \sum (y_i - \hat{y}_i)^2$

- **含义**：误差**平方后平均**
- **特点**：**惩罚大误差**，对异常值很敏感，优化常用

## 3. RMSE（均方根误差）

$RMSE=\sqrt{MSE}​$

- **含义**：MSE 开根号，**回到原量纲**
- **特点**：最常用、最直观，**可直接解释为平均误差大小**

## 4. R²（决定系数）

$R^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2}$

- **含义**：模型**解释了多少数据的方差**
- 范围：**0~1**，越接近 1 越好
- **特点**：不受量纲影响，可跨数据集比较

---

# 一句话终极背诵（背这一段）

**MAE 绝对平均，MSE 罚大误差，RMSE 还原量纲最常用，R² 看解释力（越接近 1 越好）。**
