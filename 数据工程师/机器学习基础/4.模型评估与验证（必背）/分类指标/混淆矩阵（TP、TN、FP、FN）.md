# 混淆矩阵 + TP/TN/FP/FN（面试**死记硬背版**）

先记住一句话：**Positive/Negative 是预测，True/False 是对错**

## 四个值（必须秒懂）

- **TP（真正例）**：预测正，实际正 → **猜对了正**
- **TN（真负例）**：预测负，实际负 → **猜对了负**
- **FP（假正例）**：预测正，实际负 → **误杀 / 误报**（把负说成正）
- **FN（假负例）**：预测负，实际正 → **漏报 / 漏检**（把正说成负）

### 最简记忆口诀

- **T：预测对了**
- **F：预测错了**
- **P：预测成正**
- **N：预测成负**

---

## 标准混淆矩阵结构（2 分类）

```plaintext
            实际正    实际负
预测正        TP       FP
预测负        FN       TN
```

---

## 一句话背诵

**TP 猜对正，TN 猜对负，FP 误报，FN 漏检。**
