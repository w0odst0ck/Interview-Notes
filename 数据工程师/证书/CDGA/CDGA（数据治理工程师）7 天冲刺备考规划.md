
CDGA 偏**概念识记 + 基础理解**，无复杂计算，7 天完全够用，核心抓：**数据治理框架、标准规范、流程工具、法律法规、案例常识**。

## 总原则

- 每天**3～4 小时**高效学习即可
- 重点：**官方教材 / 讲义 > 真题 / 题库 > 错题复盘**
- 不钻难题，不深究偏理论，只抓**高频考点**

# 第 1 天：整体框架搭建（入门 + 通读）

目标：知道 CDGA 考什么、全书结构

1. 快速通读**CDGA 考试大纲**
2. 浏览教材 / 讲义目录，建立知识框架
    
    - 数据治理基础概念
    - 数据治理体系与组织
    - 数据战略、数据标准、数据质量
    - 数据安全、数据生命周期、元数据
    - 法规与合规（重点：个人信息保护、数据安全法）
    
3. 做 1 套**摸底真题 / 模拟卷**
    
    - 不追求分数，只看薄弱模块
        
        用时：3h

# 第 2 天：核心模块一（治理基础 + 组织 + 战略）

重点背诵 + 理解

2. DAMA-DMBOK2 核心知识域（CDGA 必考）

4. 数据战略制定、实施路径
5. 刷对应章节选择题
    
    用时：3.5h

# 第 3 天：核心模块二（数据标准 + 数据质量）

考试**分值最高**，必须吃透

1. 数据标准：业务标准、技术标准、管理标准
2. 主数据、参考数据、元数据概念与管理
3. 数据质量 6 大维度（完整性、准确性、一致性、唯一性、及时性、有效性）
4. 数据质量监控、评估、改进流程
5. 刷题库：**数据质量 + 数据标准**专项
    
    用时：3.5h

---

# 第 4 天：核心模块三（数据安全 + 合规 + 法规）

纯记忆，性价比极高

1. 数据安全治理、分级分类、权限管理
2. 关键法规：
    
    - 《数据安全法》
    - 《个人信息保护法》
    - 《网络安全法》
    
3. 数据合规、风险评估、审计
4. 数据生命周期管理（采集 - 存储 - 使用 - 共享 - 归档 - 销毁）
    
    用时：3h

1. DMBOK2 11 大知识域（CDGA 核心）
