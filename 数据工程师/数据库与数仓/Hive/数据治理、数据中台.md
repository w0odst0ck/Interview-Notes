结合你**东华大学 AI 专业背景 + Python/SQL 基础**，以及这份**数据治理 / 数据中台岗位**要求，给你一套**7 天高强度、针对性突击学习路径**，只学岗位刚需，不搞无用拓展。

# 7 天突击学习路径（数据治理岗方向）

## Day1：数据治理体系 + 岗位核心认知

- 掌握数据治理整体框架：元数据、数据质量、数据标准、数据资产、主数据
- 理解 CDGA/CDMP 核心知识体系（不用考证，只学考点）
- 通读岗位 JD，明确每天要补的技术缺口
- 目标：能说清 “数据治理到底做什么”

## Day2：数据仓库 + 数据建模核心

- 维度建模（星型、雪花）、总线架构
- 概念 / 逻辑 / 物理模型、主题域划分
- 常用建模方法论与工具逻辑
- 目标：能独立设计一套简单业务数仓模型

## Day3：大数据技术栈快速入门（Hive/Spark/Hadoop）

- 重点：Hive SQL 语法、分区、分桶、优化
- 了解 Spark 基础与批处理场景
- 理解大数据平台整体架构
- 目标：看得懂数仓开发脚本，能写基础 Hive SQL

## Day4：数据采集 + ETL + 数据开发

- ETL 工具逻辑（DataWorks、DataX 等思想）
- API 数据采集思路
- 数据清洗、数据质量规则设计
- 结合 Python 做简单 ETL 脚本
- 目标：能完成一套小型数据同步流程

## Day5：数据资产管理 + 主数据 / 元数据

- 元数据管理：血缘、采集、展示
- 主数据、参考数据管理逻辑
- 数据资产盘点、资产目录建设
- 目标：能讲清楚企业数据资产怎么做

## Day6：项目文档 + 方案撰写 + 业务落地

- 学习数据治理项目实施方案结构
- 练习写：数据模型设计文档、数据治理方案
- 跨部门协作、需求拆解思路
- 目标：能产出岗位要求的 “高质量文档”

## Day7：AI 工具增效 + 交通运输行业场景 + 复盘整合

- 学习用 Kimi/DeepSeek 辅助写方案、查知识点
- PowerBI AI 数据分析基础
- 了解交通运输行业典型数据（客流、车辆、轨迹）
- 整体复盘：把 7 天内容串成面试话术
- 目标：能完整自我介绍 + 项目经历 + 技术问答
