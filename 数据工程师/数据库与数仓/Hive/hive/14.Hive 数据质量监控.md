# 14. Hive 数据质量监控（空值、重复、一致性、规则校验）

## 一、简要面试介绍

- **空值监控**：检查主键、关键字段是否存在 `NULL`/ 空字符串，保证数据完整性。
- **重复监控**：检查主键 / 唯一键是否重复，避免明细数据重复计算。
- **一致性监控**：跨层、跨表指标一致；关联字段类型、枚举值统一。
- **规则校验**：业务规则校验，如值域范围、长度规范、格式合法性、逻辑合理性。

## 二、综合示例（串联所有点）

```sql
-- 1. 空值检查
SELECT COUNT(*) AS null_uid FROM dwd.user_login WHERE uid IS NULL OR uid = '';

-- 2. 重复数据检查
SELECT uid,dt,COUNT(*) AS cnt 
FROM dwd.user_login 
GROUP BY uid,dt 
HAVING COUNT(*) > 1;

-- 3. 一致性检查（DWS 汇总值应等于 DWD 明细之和）
SELECT 
  a.dt,
  a.total_pv AS dws_pv,
  b.total_pv AS dwd_pv
FROM dws.user_daily a
JOIN (SELECT dt,SUM(pv) AS total_pv FROM dwd.user_login GROUP BY dt) b 
ON a.dt = b.dt
WHERE a.total_pv != b.total_pv;

-- 4. 规则校验（值域、格式、枚举）
SELECT COUNT(*) AS illegal
FROM dwd.user_login
WHERE 
  login_time > '2025-12-31'  -- 日期越界
  OR length(phone) != 11      -- 手机号长度不合法
  OR status NOT IN ('0','1'); -- 枚举值非法
```

## 三、一句话总结

Hive 数据质量通过**空值校验保完整、重复校验保唯一、一致性校验保准确、规则校验保合规**，实现数仓数据可信任、可复用。
