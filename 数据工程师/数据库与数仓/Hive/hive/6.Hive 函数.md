## 一、简要面试介绍

1. **内置函数**
    
    Hive 自带常用函数，直接使用即可，主要分：

- 字符串：`concat`/`substr`/`split`/`nvl`
- 日期：`date_format`/`date_add`/`datediff`/`from_unixtime`
- 聚合：`count`/`sum`/`avg`/`max`/`min`/`row_number`
- 条件：`case when`/`if`/`coalesce`

2. **UDF（用户自定义函数）**
    
    一进一出，对单行数据做处理，如自定义脱敏、格式转换。
    
3. **UDAF（用户自定义聚合函数）**
    
    多进一出，多行聚合为一个值，如自定义去重统计、分位数计算。
    
4. **UDTF（用户自定义表生成函数）**
    
    一进多出，一行拆成多行 / 多列，如 JSON 展开、数组炸裂，类似 `explode`。

## 二、综合示例（串联）

```sql
-- 内置函数 + 侧视图 UDTF 炸裂
SELECT
  uid,
  -- UDF 风格内置函数
  nvl(name, 'unknown') AS user_name,
  -- 聚合函数 UDAF
  count(pv) AS total_pv,
  -- 日期函数
  date_format(ts, 'yyyy-MM-dd') AS dt
FROM
  ods.user_log
-- UDTF：数组拆多行
LATERAL VIEW explode(split(page_list, ',')) tmp AS pv
WHERE
  dt = '2025-04-01'
GROUP BY
  uid, nvl(name, 'unknown'), date_format(ts, 'yyyy-MM-dd');
```

开发流程简述：

1. 写 Java/Scala 实现 `UDF/UDAF/UDTF` 接口
2. 打包成 jar 上传 HDFS
3. `add jar xxx.jar` 注册函数
4. `create temporary function func_name as '包名.类名'` 使用

## 三、一句话总结

Hive 函数分为**内置函数直接用**，**UDF 单行处理、UDAF 聚合计算、UDTF 行列转换**，通过开发 + 注册即可扩展数据处理能力。
