# Hive DDL / DML 核心语法（面试必背）

## 一、DDL（数据定义语言）

负责：库、表、分区的**创建、修改、删除**

### 1. 库操作

```sql
CREATE DATABASE IF NOT EXISTS db_name;
USE db_name;
ALTER DATABASE db_name SET DBPROPERTIES(...);
DROP DATABASE IF EXISTS db_name CASCADE;
```

### 2. 建表（标准生产语法）

```sql
CREATE EXTERNAL TABLE IF NOT EXISTS tb_name(
    id    INT,
    name  STRING,
    ts    TIMESTAMP
)
PARTITIONED BY (dt STRING)
CLUSTERED BY (id) INTO 10 BUCKETS
ROW FORMAT DELIMITED FIELDS TERMINATED BY ','
STORED AS ORC
TBLPROPERTIES ("orc.compress"="SNAPPY");
```

### 3. 修改表

```sql
ALTER TABLE tb_name RENAME TO new_tb_name;
ALTER TABLE tb_name ADD COLUMNS (age INT);
ALTER TABLE tb_name CHANGE COLUMN id uid BIGINT;
ALTER TABLE tb_name SET TBLPROPERTIES(...);
```

### 4. 删除表

```sql
DROP TABLE IF EXISTS tb_name;
TRUNCATE TABLE tb_name;
```

### 5. 分区管理（高频）

```sql
ALTER TABLE tb_name ADD PARTITION (dt='2025-01-01');
ALTER TABLE tb_name DROP PARTITION (dt='2025-01-01');
ALTER TABLE tb_name PARTITION (dt='2025-01-01') RENAME TO PARTITION (dt='2025-01-02');
MSCK REPAIR TABLE tb_name;  -- 修复分区
SHOW PARTITIONS tb_name;   -- 查看分区
```

## 二、DML（数据操作语言）

负责：数据**增、删、改、查**

### 1. 查询（最核心）

```sql
SELECT id, name FROM tb_name WHERE dt='2025-01-01' GROUP BY id;
```

### 2. 插入数据

```sql
-- 覆盖写入
INSERT OVERWRITE TABLE tb_name PARTITION(dt='2025-01-01')
SELECT id, name FROM tmp_tb;

-- 追加写入
INSERT INTO TABLE tb_name PARTITION(dt='2025-01-01')
VALUES (1,'zhangsan'), (2,'lisi');
```

### 3. 更新 & 删除（仅 ACID 表）

```sql
UPDATE tb_name SET name='zs' WHERE id=1;
DELETE FROM tb_name WHERE id=1;
```

### 4. 数据加载

```sql
LOAD DATA INPATH '/tmp/data.txt' OVERWRITE INTO TABLE tb_name PARTITION(dt='2025-01-01');
```

# 综合示例（串联所有语法，面试直接说）

```sql
-- 建库
CREATE DATABASE IF NOT EXISTS dwd;
USE dwd;

-- 建外部分区表
CREATE EXTERNAL TABLE user_login(
    uid BIGINT,
    ip STRING
) PARTITIONED BY (dt STRING)
STORED AS ORC;

-- 新增分区
ALTER TABLE user_login ADD PARTITION (dt='2025-01-01');

-- 插入数据
INSERT OVERWRITE TABLE user_login PARTITION(dt='2025-01-01')
SELECT uid, ip FROM ods.user_raw WHERE dt='2025-01-01';

-- 查询
SELECT uid, COUNT(*) FROM user_login WHERE dt='2025-01-01' GROUP BY uid;

-- 修改字段
ALTER TABLE user_login CHANGE COLUMN ip client_ip STRING;

-- 删除分区
ALTER TABLE user_login DROP PARTITION (dt='2025-01-01');
```

# 一句话总结

**DDL 管理库表分区结构，DML 负责数据增删改查，生产核心是外部分区表 + 分区管理 + 覆盖插入 + 分区查询。**
