- PyMilvus/Go/Java/C++ SDK、REST API、与 Embedding 模型 / ETL / 数据中台对接
## 一、面试极简知识点

- **PyMilvus**：Python 官方 SDK，最常用，适合原型、数据处理、RAG。
- **Go/Java/C++ SDK**：服务端 / 高性能场景，生产微服务对接。
- **REST API**：跨语言、跨平台调用，轻量服务对接。
- **Embedding 模型对接**：与 BERT、OpenAI、文心一言等向量模型串联，生成向量写入 Milvus。
- **ETL 对接**：从 MySQL/Kafka/Hive 抽取数据 → 向量化 → 写入 Milvus。
- **数据中台对接**：作为向量检索引擎，接入中台统一元数据、权限、调度。

## 二、综合示例

企业文档检索系统：

1. 用 **PyMilvus** 连接 Milvus；
2. 从中台 ETL 同步文档数据；
3. 调用 Embedding 模型生成向量；
4. 写入 Collection；
5. 提供 REST API 给前端做检索服务。

## 三、一句话总结

通过多语言 SDK / REST API 实现开发接入，对接 Embedding 模型、ETL、数据中台完成向量数据全流程生产与消费。
