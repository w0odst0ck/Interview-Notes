# 聚合与高级函数 面试精简版（数据中台 / 治理必考）

## 一、核心知识点（必背）

### 1. 聚合函数

- **COUNT()**：计数
    
    - `COUNT(*)`：统计所有行（包含 NULL）
    - `COUNT(字段)`：统计字段非空行
    - **COUNT (DISTINCT 字段)**：**去重计数**，统计唯一值数量（高频）
    

### 2. 条件判断函数

- **CASE WHEN**：SQL 里的**if-else**，通用、最常用
    
    语法：`CASE WHEN 条件1 THEN 结果1 WHEN 条件2 THEN 结果2 ELSE 其他 END`

### 3. 空值替换函数

- **COALESCE (字段，默认值)**：**通用标准**，返回第一个非 NULL 值，可传多个参数
- **NVL (字段，默认值)**：Oracle/Spark SQL 专用，2 个参数
- **IFNULL (字段，默认值)**：MySQL 专用，2 个参数

### 4. 常用高阶函数

- **日期函数**：日期加减、获取年月日、日期格式化、时间戳转换
- **字符串函数**：拼接、截取、去空格、替换、转大小写、分割
- **数值函数**：四舍五入、向上 / 下取整、取绝对值、取余

## 二、综合示例（串联所有）

### 表：用户订单表 order

|user_id|order_amount|create_date|status|
|---|---|---|---|
|1001|NULL|2025-01-01|1|
|1001|200.5|2025-01-01|2|
|1002|99.0|2025-01-02|1|
|1003|300.8|2025-01-03|2|

### 综合查询 SQL

需求：统计每个用户的订单数、去重用户数、空金额补 0、订单状态翻译、订单月份、订单金额取整

```sql
SELECT
  user_id,
  -- 标准计数
  COUNT(order_id) AS order_cnt,
  -- 去重计数（面试高频）
  COUNT(DISTINCT user_id) AS user_cnt,
  -- 空值替换（通用）
  COALESCE(order_amount, 0) AS real_amount,
  -- 条件判断（状态翻译）
  CASE 
    WHEN status = 1 THEN '待支付'
    WHEN status = 2 THEN '已支付'
    ELSE '其他'
  END AS status_name,
  -- 日期函数：取月份
  SUBSTRING(create_date, 1, 7) AS order_month,
  -- 数值函数：四舍五入取整
  ROUND(order_amount) AS amount_round
FROM order
GROUP BY user_id;
```

## 三、一句话总结

COUNT (DISTINCT) 做去重统计，CASE WHEN 做条件分支，COALESCE/NVL/IFNULL 处理空值，搭配日期、字符串、数值函数完成数据清洗与转换。

### 总结

1. **COUNT(DISTINCT)**：统计唯一值，去重计数必备
2. **CASE WHEN**：数据打标、状态转换必备
3. **空值函数**：避免 NULL 影响计算，优先用 COALESCE（通用）
4. **三类函数**：数据中台 / 治理**数据清洗、指标加工**核心工具
