# 窗口函数 面试精简版（数据中台 / 治理必考）

## 一、核心知识点（必背）

1. **窗口函数**

- 不分组、不合并行，**保留原表所有行**，同时计算聚合 / 排名
- 语法：`函数() OVER (PARTITION BY 分组字段 ORDER BY 排序字段)`

2. **ROW_NUMBER()**

- 连续排名：1,2,3,4
- **同分不同名**，不会重复

3. **RANK()**

- 跳跃排名：1,2,2,4
- **同分同名，跳过名次**

4. **DENSE_RANK()**

- 连续密集排名：1,2,2,3
- **同分同名，不跳名次**

5. **SUM/AVG/COUNT() OVER()**

- 分组内**累计求和、平均值、计数**
- 可做**分组累计、滑动计算**

---

## 二、综合示例（串联所有）

### 表：学生成绩表 score

|student_id|class|score|
|---|---|---|
|1|1 班|90|
|2|1 班|90|
|3|1 班|85|
|4|2 班|88|
|5|2 班|92|

### 1. 排名三剑客

```sql
SELECT
  student_id,
  class,
  score,
  ROW_NUMBER() OVER(PARTITION BY class ORDER BY score DESC) AS rn,
  RANK() OVER(PARTITION BY class ORDER BY score DESC) AS rk,
  DENSE_RANK() OVER(PARTITION BY class ORDER BY score DESC) AS drk
FROM score;
```

### 2. 聚合窗口函数

```sql
SELECT
  student_id,
  class,
  score,
  -- 班级总分
  SUM(score) OVER(PARTITION BY class) AS class_sum,
  -- 班级平均分
  AVG(score) OVER(PARTITION BY class) AS class_avg,
  -- 班级内累计分数
  SUM(score) OVER(PARTITION BY class ORDER BY score DESC) AS cum_sum
FROM score;
```

---

## 三、一句话总结

窗口函数不合并行数，ROW_NUMBER 连续不重复、RANK 跳跃、DENSE_RANK 密集，SUM/AVG/COUNT () OVER 可做分组内聚合与累计计算。

### 总结

1. **排名函数**：取 TopN、去重、排序必备
2. **聚合窗口**：分组累计、同组对比必备
3. **中台高频场景**：拉链表、最新状态、TopN 统计
