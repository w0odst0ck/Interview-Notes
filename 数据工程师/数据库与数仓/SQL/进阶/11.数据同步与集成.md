- INSERT OVERWRITE / INSERT INTO、数据去重、增量同步 SQL
# 11. 数据同步与集成（面试极简背诵版）

## 一、核心知识点

1. **INSERT OVERWRITE**

- 覆盖写入：先删除目标数据，再插入新数据
- 数仓日常同步**最常用**，适合分区表每日重跑

2. **INSERT INTO**

- 追加写入：直接在末尾添加数据
- 适合日志类、实时写入，**容易产生重复**

3. **数据去重**

- 常用：`ROW_NUMBER() OVER(PARTITION BY ... ORDER BY ...)`
- 保证主键唯一，避免重复数据

4. **增量同步**

- 只同步**新增 / 变化数据**，不扫全量
- 按时间、日志号、版本号筛选

---

## 二、综合示例

### 1. 全量覆盖同步（OVERWRITE）

```sql
INSERT OVERWRITE TABLE dwd_order PARTITION(dt='2025-04-01')
SELECT order_id, user_id, amount, status
FROM ods_order
WHERE dt='2025-04-01';
```

### 2. 数据去重（窗口函数去重，标准写法）

```sql
INSERT OVERWRITE TABLE dwd_order PARTITION(dt='2025-04-01')
SELECT 
  order_id, user_id, amount, status
FROM (
  SELECT 
    *,
    ROW_NUMBER() OVER(PARTITION BY order_id ORDER BY update_time DESC) rn
  FROM ods_order
  WHERE dt='2025-04-01'
) t
WHERE rn = 1;
```

### 3. 增量同步（只同步当天新增）

```sql
INSERT INTO TABLE dwd_order
SELECT order_id, user_id, amount, status
FROM ods_order
WHERE dt = '2025-04-01'
  AND create_time >= '2025-04-01 00:00:00';
```

---

## 三、一句话总结

INSERT OVERWRITE 覆盖重跑、INSERT INTO 追加写入，用窗口函数做数据去重，按时间 / 版本实现高效增量同步。
