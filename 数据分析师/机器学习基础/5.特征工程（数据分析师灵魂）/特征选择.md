# 特征工程 + 特征选择（数据分析师灵魂）｜面试极简背诵版

## 一、特征工程是什么（一句话）

**从原始数据→构造 / 清洗 / 转换→让模型更好用的特征，是数据分析 / 建模效果的核心灵魂。**

包含四步：

1. **数据清洗**（缺失值、异常值、重复值）
2. **特征构造**（组合、统计、时序、交叉特征）
3. **特征转换**（归一化、标准化、分桶、编码）
4. **特征选择**（去掉无用 / 冗余 / 噪声特征，防过拟合、提速）

---

# 二、特征选择三大类方法（必考）

## 1. 过滤法 Filter

- **不依赖模型**，单独算统计量
- 方法：
    
    - 方差筛选（去掉低方差）
    - 相关系数（Pearson、Spearman）
    - 卡方检验、互信息
    
- **优点**：快、通用
- **缺点**：不考虑特征间交互

## 2. 包装法 Wrapper

- **用模型效果来选特征**
- 方法：**递归特征消除 RFE**
- 思路：不断删最差特征，看模型分数
- **优点**：效果好
- **缺点**：极慢、易过拟合

## 3. 嵌入法 Embedded

- **训练过程中自动选**
- 方法：
    
    - **L1 正则（Lasso）系数为 0**
    - 树模型特征重要性
    
- **优点**：兼顾速度与效果，最常用
- **实战首选**

---

# 三、一句话终极背诵

**特征工程是建模灵魂；特征选择分三类：Filter 快、Wrapper 准但慢、Embedded（L1 / 树重要性）最实用。**
