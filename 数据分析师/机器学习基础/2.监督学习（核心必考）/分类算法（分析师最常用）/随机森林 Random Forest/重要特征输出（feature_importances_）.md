# 随机森林：特征重要性（feature_importances_）｜面试极简版

## 一、是什么

随机森林**自带输出**：每个特征**对模型预测的贡献度**，数值越大越重要。

## 二、计算原理（背这一句）

**特征重要性 = 这个特征在所有树中，带来的「纯度提升总和」**

- 分类：基尼 / 熵 减少越多 → 越重要
- 回归：MSE 减少越多 → 越重要

## 三、特点（必考）

1. **自动归一化**：总和 = 1
2. 不用手动计算，**直接调用**
3. 可用于**特征筛选**（删掉不重要特征）
4. 树模型自带，**不需要特征缩放**

## 四、一句话总结

**用分裂时的纯度提升总和衡量重要性，数值越大，特征越关键。**
