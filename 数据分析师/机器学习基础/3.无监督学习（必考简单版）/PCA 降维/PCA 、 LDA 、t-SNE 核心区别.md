# PCA / LDA /t-SNE 核心区别（速记）

## 1. PCA（主成分分析）

- **类型**：无监督降维
- **目标**：**保留最大方差**
- **是否用标签**：不用标签
- **特点**：线性、快、通用
- **用途**：去冗余、降噪、可视化、预处理
- **缺点**：无业务解释性

## 2. LDA（线性判别分析）

- **类型**：有监督降维
- **目标**：**类间距离最大，类内距离最小**
- **是否用标签**：**必须用标签**
- **特点**：线性、带监督、分类更友好
- **用途**：分类前降维，比 PCA 更利于分类
- **维度上限**：最多降为 **类别数 - 1**

## 3. t-SNE

- **类型**：无监督降维
- **目标**：**保留局部结构**
- **是否用标签**：不用
- **特点**：**非线性、效果好、极慢**
- **用途**：**只用来可视化（2D/3D 画图）**
- **不能做**：不能当特征输入模型（无映射函数）

# 一句话终极区分（背这个就够）

- **PCA：无监督，保方差，线性快，通用降维**
- **LDA：有监督，保分类信息，线性，分类专用**
- **t-SNE：无监督，保局部，非线性慢，**只画图**
