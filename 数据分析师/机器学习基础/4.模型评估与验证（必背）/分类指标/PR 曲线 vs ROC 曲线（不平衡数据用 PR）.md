# PR 曲线 vs ROC 曲线（面试必问：不平衡数据用 PR）

给你**最精简、直接背**的版本：

## 一、ROC 曲线

- 纵：**TPR（召回率）**
- 横：**FPR（误报率）**
- 评价**整体正负样本区分能力**
- 优点：**不受类别不平衡影响**（看起来总是很好）
- 缺点：**不平衡数据时会 “骗人”**，看起来很高，实际很差

## 二、PR 曲线（Precision-Recall）

- 纵：**精确率 Precision**
- 横：**召回率 Recall**
- **只关注正样本**
- 优点：**极度敏感于不平衡数据**，好坏一眼看穿
- 缺点：平衡数据不如 ROC 稳定

---

# 三、核心结论（背这句就够）

- **平衡数据**：用 **ROC / AUC**
- **极度不平衡数据（欺诈、疾病、异常检测）**：必须用 **PR 曲线 / AP**
    
    → ROC 会虚高，**PR 更真实、更严格**

---

# 一句话终极背诵

**ROC 看整体区分，不平衡数据会虚高；PR 只盯正样本，更严格真实，不平衡数据必选 PR。**
