# 数据分析师必备深度学习基础（面试版）

## 一、核心基础概念（必背）

1. **深度学习 vs 机器学习**
    
    - 区别：特征工程是否自动学习
    - 适用场景：图像、文本、序列、高维数据
    
2. **神经元与感知机**
    
    - 输入、权重、偏置、激活函数
    
3. **前向传播 & 反向传播**
    
    - 前向：算预测值
    - 反向：算梯度、更新权重（链式法则）
    
4. **损失函数 Loss**
    
    - 分类：交叉熵 Cross Entropy
    - 回归：MSE、MAE
    
5. **优化器 Optimizer**
    
    - SGD、Adam（最常问）
    - 学习率、batch、epoch 含义
    
6. **过拟合 & 解决方法**
    
    - Dropout
    - 早停 Early Stopping
    - L1/L2 正则
    - 数据增强
    

## 二、激活函数（必问）

- **Sigmoid**：0~1，二分类输出，梯度消失
- **Tanh**：-1~1
- **ReLU**：最常用，解决梯度消失
- **LeakyReLU**：解决死 ReLU

## 三、必备网络结构（懂概念 + 适用场景）

### 1. 全连接层 DNN / MLP

- 表格数据、传统特征建模
- 数据分析师**最常用**的深度学习 baseline

### 2. CNN 卷积神经网络

- 核心：**卷积、池化、padding、stride**
- 作用：提取局部特征、权值共享
- 场景：图像、时序、NLP 局部特征

### 3. RNN / LSTM / GRU

- 处理**序列数据**：时间序列、文本、用户行为序列
- LSTM 解决 RNN 梯度消失
- 门机制：输入门、遗忘门、输出门（懂概念即可）

### 4. Transformer / Attention（高频！）

- **自注意力机制 Self-Attention**
- 场景：NLP、推荐、时间序列预测
- 知道 BERT、GPT 都是基于 Transformer

## 四、NLP 基础（数据分析岗高频）

- 词向量：Word2Vec、Embedding
- Token、分词
- 文本分类、情感分析、语义匹配
- 大模型基础：RAG、Prompt 工程（**OD 非常爱问**）

## 五、计算机视觉基础（简单了解）

- 图像分类、目标检测概念
- 卷积、池化作用
- 常见模型：ResNet（懂思想即可）

## 六、训练与工程常识（必问）

- 训练流程：数据→划分→构建模型→编译→训练→评估
- Batch、Epoch、Iteration 区别
- 梯度消失 / 爆炸
- 标准化、归一化
- 评价指标：Acc、Precision、Recall、F1、AUC、Perplexity

## 七、数据分析岗**必考实战方向**

1. **深度模型在数据分析中的应用**
    
    - 用户行为序列预测（LSTM/Transformer）
    - 文本特征提取（BERT 嵌入）
    - 深度推荐模型（DeepFM、Wide&Deep）
    
2. **传统机器学习 vs 深度学习怎么选**
    
    - 表格小数据：树模型优先
    - 文本 / 序列 / 图像：深度学习
    
3. **RAG 原理**（你做过项目，必问）
    
    - 检索增强生成
    - 向量数据库、Embedding、Chunk 切分
    

## 八、工具与框架（会用即可）

- TensorFlow / Keras（简单易懂）
- PyTorch（主流）
- Hugging Face Transformers
- 会：**加载数据→搭建模型→训练→评估**即可
